Version history

Release notes and changes for each version of Check Your Backup.

v1.0.7 June 9, 2026 Latest

UI refresh, Info menu and Windows build availability.

  • Redesigned header with larger branding and clearer read-only notice.
  • Improved About dialog and summary statistics row with status icons.
  • Replaced Help menu with Info: Quick Start, corrupted-file guide, read-only explainer, support and website links.
  • Updated PayPal support link.
  • Windows 10/11 portable build now available for download.
v1.0.6 June 9, 2026

Trust, clarity and professionalism — without changing how analysis works.

Read-only assurance

  • Always-visible notice in the app: Check Your Backup never modifies, deletes, moves or repairs your files.
  • PDF cover page includes the same read-only explanation and the tagline Verify before you trust your backup.

Clearer for new users

  • Compact three-step guide in the main window: Select a folder → Analyze → Review detected issues.
  • New Help → About window with version, website and optional support link.

Diagnostic-only results

  • Removed Health Score, Backup Readiness Score and qualitative status labels (EXCELLENT, GOOD, etc.).
  • Reports and UI now show objective counts only: files analyzed, Healthy, Corrupted, Suspicious, Skipped and data analyzed.
v1.0.5 June 8, 2026

Minimalist utility redesign — one screen, one workflow.

  • Single integrity-check workflow: select folder, analyze, view results.
  • Removed Backup Validation from the app UI, plus tabs, dashboards and help panels.
  • Progress bar always visible; file table is the main focus of the window.
v1.0.0 June 7, 2026

First public release.

Integrity Check

  • Images: JPG, TIFF, PNG.
  • RAW: CR2, NEF, ARW, DNG.
  • Archives: ZIP, including analysis of images inside the archive.
  • Documents: PDF, DOCX, XLSX, PPTX.
  • Each file is classified as Healthy, Corrupted, Suspicious, Unreadable or Skipped.

Backup Validation

  • Compares an original folder with a backup (size first, then SHA-256).
  • Reports Validated, Missing in Backup, Different Size, Different Content and Extra in Backup.
  • Overall Backup Health score.

Reports & more

  • Corporate PDF report and CSV export, plus a dedicated Backup Validation PDF.
  • Per-user analysis history.
  • Multithreaded analysis with a responsive progress bar.
  • Guided onboarding: how-to steps, mode descriptions and a status legend.
  • Automatic exclusions of irrelevant files (program reports, system, temporary and hidden files).

Packaging

  • Portable, double-click builds for Windows 10/11 and macOS (Intel / Apple Silicon).
  • No Python, no admin rights, no extra dependencies. Windowed (no console).
  • Reports are written to the selected folder; nothing is written to system locations.
